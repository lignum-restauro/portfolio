# 🏛️ LIGNUM | Engenharia de Restauro

Este repositório armazena a infraestrutura digital da **Lignum**, especializada no restauro de esquadrias monumentais e preservação de patrimônio histórico.

### 🪵 O Projeto
A Landing Page foi concebida como um portfólio de alta fidelidade técnica, apresentando obras icônicas como o **Museu Nacional** e o **Parque Lage**. O design prioriza a elegância e o respeito à arquitetura clássica.

### 🛠️ Especificações Técnicas
Desenvolvimento focado em performance e segurança da informação:
- **Tecnologias:** HTML5 e CSS3 moderno (sem frameworks pesados).
- **Tipografia:** Cinzel (Clássica) & Montserrat (Minimalista).
- **Arquitetura:** Deploy estático via GitHub Pages.

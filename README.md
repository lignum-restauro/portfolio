# LIGNUM | Portfólio de Restauro e Engenharia

Este repositório contém o código-fonte da Landing Page institucional da **Lignum**, focada na apresentação de portfólio de alta complexidade técnica em restauro de patrimônio histórico.

## 🛠️ Tecnologias Utilizadas
- **Linguagens:** HTML5, CSS3 (Design Responsivo)
- **Hospedagem:** GitHub Pages
- **Estética:** Design Editorial Minimalista (Paleta: Off-white, Verde Floresta e Dourado)

## 🏛️ Projetos em Destaque
- Museu Nacional (RJ)
- Parque Lage (RJ)
- Fiocruz (RJ)
- Estação Ferroviária de Taubaté (SP)
